# Principes Techniques Avancés

## Contraintes non négociables
- Séparation génération / curriculum / admin / rendu.
- Compatibilité rétroactive obligatoire (GM07 / GM08).
- Pas de refactor hors périmètre validé.
- Déterminisme à chaque étape du pipeline.

## Validation
- Erreurs explicites (HTTP 4xx/5xx).
- Aucun fallback silencieux.
