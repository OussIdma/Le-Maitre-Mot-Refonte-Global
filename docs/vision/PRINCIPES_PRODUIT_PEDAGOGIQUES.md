# Principes Produit & Pédagogiques

## Élève
- Même entrée → même type de sortie.
- Aucun message technique exposé.
- Comportement stable et explicable.

## Enseignant / Admin
- Chaque action produit un effet prévisible.
- Aucune configuration implicite.
- Les règles sont compréhensibles sans lire le code.

## Inacceptable
- Chapitre visible mais cassé.
- Exercice incohérent avec le chapitre.
- Erreur causée par une configuration admin.
